الخريطة 1: جمع الأعداد من 1 إلى N

🟢 البداية
⬇️
📥 أدخل N
⬇️
🔧 i = 1, sum = 0
⬇️
💎 هل i > N ؟
⬇️
  ↩️ (لا)  →   sum = sum + i  →   i = i + 1  →   (عود للأعلى إلى 💎)
⬇️
  ✅ (نعم)  →   📤 اطبع sum
⬇️
🔴 النهاية

الخريطة 2: تحديد زوجي أو فردي

🟢 البداية
⬇️
📥 أدخل X
⬇️
💎 هل X mod 2 = 0 ؟
⬇️
  ✅ (نعم)  →   📤 اطبع "زوجي"
⬇️
  ↩️ (لا)  →   📤 اطبع "فردي"
⬇️
🔴 النهاية

الخريطة 3: مضاعفة العدد حتى يصل إلى 100

🟢 البداية
⬇️
🔧 y = 5
⬇️
💎 هل y < 100 ؟
⬇️
  ✅ (نعم)  →   🔧 y = y × 2  →   (عود للأعلى إلى 💎)
⬇️
  ↩️ (لا)  →   📤 اطبع y
⬇️
🔴 النهاية

الخريطة 4: عملية شرطية (الطرح تحت الشرط)

🟢 البداية
⬇️
🔧 a = 10, b = 3
⬇️
💎 هل a > b ؟
⬇️
  ✅ (نعم)  →   🔧 a = a - b
⬇️
  ↩️ (لا)  →   (لا تغيير)
⬇️
📤 اطبع a
