📌 تمرير المعاملات إلى الإجراءات الفرعية (Sub)

فهم تطابق العدد والنوع - إرجاع عدة قيم - مثال الكتاب ص95 (شكل 6-3)

📖 كيف يتم تمرير المتغيرات إلى الإجراء الفرعي؟

🔹 قواعد أساسية

  • تطابق العدد: يجب أن يكون عدد المعاملات في الاستدعاء مساوياً لعدد المعاملات في تعريف الإجراء.
  • تطابق النوع: يجب أن يتوافق نوع كل معامل (Integer, String, Single, ...) مع نوع المعامل المُعرّف.
  • إرجاع عدة قيم: يمكن للإجراء الفرعي أن يُعيد قيماً متعددة عبر المعاملات (لأن التغيير داخل الإجراء يؤثر على المتغير الأصلي).
  • لا يوجد ByVal / ByRef: في منهجنا، تمرير المعاملات يتم بالطريقة الافتراضية (تمرير بالعنوان).

🔹 مثال من الكتاب (ص95 – شكل 6-3)

الإجراء الفرعي calc يستقبل قيمتين (x, y) ويعيد أربع نتائج عبر المعاملات s1, s2, s3, s4.

Private Sub Command1_Click()
    Dim v1, v2 As Integer
    Dim s1, s2, s3 As Integer
    Dim s4 As Single
    v1 = Val(Text1.Text)
    v2 = Val(Text2.Text)
    Call calc(v1, v2, s1, s2, s3, s4)
    Print s1, s2, s3, s4
End Sub

Private Sub calc(x As Integer, y As Integer, _
             s1 As Integer, s2 As Integer, _
             s3 As Integer, s4 As Single)
    s1 = x + y
    s2 = x - y
    s3 = x * y
    s4 = x / y
End Sub

ملاحظة: المتغيرات s1, s2, s3, s4 تُستخدم لاستقبال النتائج، لذا يجب أن تتطابق أنواعها مع العمليات.

💡 تذكير: عند استدعاء الإجراء باستخدام Call، تُكتب المعاملات بين قوسين. يمكن الاستغناء عن Call وكتابة الاسم فقط مع القوسين.

السؤال 1: تطابق العدد

إذا كان الإجراء الفرعي مُعرّفاً كالتالي:
Private Sub process(a As Integer, b As Integer, c As Integer)
أي من عبارات الاستدعاء التالية صحيحة؟

Call process(5, 10)
process 5, 10, 15
Call process(5, 10, 15)
process(5, 10)

السؤال 2: تطابق النوع

الإجراء التالي:
Private Sub calculate(x As Integer, y As Single)
أيهما يعتبر استدعاء صحيحاً (مع الأخذ بعين الاعتبار أن Val تحول النص إلى رقم)؟

Call calculate(Text1.Text, Text2.Text)
Call calculate(Val(Text1.Text), Val(Text2.Text))
calculate(Text1.Text, Text2.Text)
calculate(Val(Text1.Text), Text2.Text)

السؤال 3: إرجاع عدة قيم

في مثال الكتاب (ص95)، كيف يتم إرجاع النتائج الأربع (الجمع، الطرح، الضرب، القسمة) إلى المتغيرات في الحدث Command1_Click؟

باستخدام Return
بتمرير المتغيرات (s1,s2,s3,s4) كمعاملات للإجراء الفرعي
بواسطة متغير عمومي (Public)
بواسطة دالة Function

السؤال 4: تصحيح الخطأ

نظراً للكود التالي:
Private Sub test(a As Integer, b As Integer, c As Integer)
وأثناء الاستدعاء: Call test(5, 10)
