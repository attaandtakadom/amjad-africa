📊 جدول المقارنة بين Sub و Function

وجه المقارنةالإجراء الفرعي (Sub)الدالة (Function)
🔹 الهدفتنفيذ مهمة (طباعة، حفظ، عرض)تنفيذ مهمة وترجع قيمة واحدة
🔹 كلمة الإعلانSub أو Private SubFunction أو Private Function
🔹 إنهاء الكتلةEnd SubEnd Function
🔹 إرجاع قيمة❌ لا يرجع قيمة✅ يرجع قيمة (يُحدد نوعها)
🔹 طريقة الاستدعاءCall MySub(5) أو MySub 5x = MyFunction(5) (ضمن تعبير)
🔹 مكان النتيجةتُخرج عبر ByRef أو متغيرات عموميةتُحدد عبر اسم الدالة: FunctionName = value

💻 أمثلة عملية مع الكود

📌 مثال 1: إجراء فرعي (Sub) - لا يُرجع قيمة

VB.NET / VBA
' إجراء لحساب المجموع وطباعته فقط (لا يُرجع شيئاً)
Private Sub PrintSum( a As Integer,  b As Integer)
    Dim result As Integer
    result = a + b
    MsgBox("المجموع هو: " & result)
End Sub

' كيفية استدعائه:
PrintSum(5, 3)   ' يظهر رسالة "المجموع هو: 8"

📌 مثال 2: دالة (Function) - تُرجع قيمة

VB.NET / VBA
' دالة تحسب المجموع وتُرجع القيمة
Private Function CalculateSum( a As Integer,  b As Integer) As Integer
    CalculateSum = a + b   ' تعيين القيمة العائدة
End Function

' كيفية استدعائها:
Dim total As Integer
total = CalculateSum(5, 3)   ' total = 8
MsgBox("المجموع هو: " & total)

📌 مثال 3: مقارنة مباشرة بينهما

VB.NET / VBA
' Sub: يُنفذ المهمة دون إرجاع
Private Sub GreetUser( name As String)
    MsgBox("مرحباً " & name)
End Sub

' Function: يُنفذ المهمة ويُرجع نصاً
Private Function GetGreeting( name As String) As String
    GetGreeting = "مرحباً " & name
End Function

' الاستخدام:
GreetUser("أحمد")                    ' يظهر رسالة فوراً

Dim msg As String
msg = GetGreeting("أحمد")            ' يُخزن النص في المتغير
MsgBox(msg)                          ' ثم يُعرض

⚠️ نقاط مهمة تذكرها

✅ متى تستخدم Sub؟
عندما تريد تنفيذ أمر (طباعة، حفظ، تغيير لون، فتح نافذة) ولا تحتاج إلى قيمة ناتجة.
✅ متى تستخدم Function؟
عندما تحتاج إلى حساب قيمة وإعادة استخدامها (مثل: جمع، ضرب، تنسيق نص، تحويل عملة).
⚠️ تذكير مهم:
• الدالة يجب أن تُرجع قيمة واحدة عبر اسم الدالة.
• الإجراء الفرعي لا يمكنه إرجاع قيمة مباشرة.
• تنتهي الدالة بـ End Function والإجراء بـ End Sub.

✏️ تمارين شاملة (اختيار من متعدد وصح/خطأ)

أجب عن الأسئلة التالية. لكل سؤال زر "تحقق" يظهر لك التصحيح مباشرة.
