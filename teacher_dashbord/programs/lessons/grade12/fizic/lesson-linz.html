أولاً: قانون المحولات الكهربائية

Vp / Vs = Np / Ns
Ip / Is = Ns / Np
القانون العام للمحولات الكهربائية

حيث أن:

  • Vp : جهد الملف الابتدائي
  • Vs : جهد الملف الثانوي
  • Np : عدد لفات الملف الابتدائي
  • Ns : عدد لفات الملف الثانوي
  • Ip : تيار الملف الابتدائي
  • Is : تيار الملف الثانوي

محول رفع الجهد

Ns > Np
Vs > Vp
Is < Ip

محول خفض الجهد

Ns < Np
Vs < Vp
Is > Ip

ثانياً: نقل القدرة وتقليل الفقد

Pf = I²R = (P/V)² × R
قدرة الفقد = مربع التيار × المقاومة

طرق تقليل الفقد في القدرة:

1. رفع الجهد الكهربائي

Pf ∝ 1/V²
مضاعفة الجهد تقلل الفقد إلى الربع

2. تقليل مقاومة الموصلات

R = ρL/A
زيادة مساحة المقطع تقلل المقاومة

ثالثاً: قانون لينز (Lenz's Law)

ε = -N dΦ/dt
القوة الدافعة الكهربائية الحثية

نص القانون: "يكون اتجاه التيار الحثي بحيث يقاوم التغير في التدفق المغناطيسي الذي سببه"

تطبيقات قانون لينز:

  • تحديد اتجاه التيار الحثي
  • الفرامل المغناطيسية
  • الرفع المغناطيسي (ماجليف)
  • توليد التيارات الدوامية

رابعاً: قاعدة حفظ الطاقة

Pin = Pout + Ploss
القدرة الداخلة = القدرة الخارجة + الفقد

في المحول المثالي: Pin = Pout → VpIp = VsIs

كفاءة المحول:

η = (Pout / Pin) × 100%
النتيجة: 0 / 10

الملف الثانوي عدد لفاته 50 لفة
المطلوب: حساب جهد الثانوي وتيار الثانوي (إذا كان تيار الابتدائي 2A)

المطلوب: حساب قدرة الفقد في الخط
